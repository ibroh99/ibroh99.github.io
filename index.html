ALICE - Amikom Learning Center

Project Brief:

Previously, lecturers and students used Google Classroom and the existing Amikom e-learning for coursework management. ALICE is here to adopt the features in Google Classroom, combines it with the existing Amikom e-learning application features, and adds many other extra features. Thus, students and lecturers can use just one application to do everything.

Note: This application is only made in order to work on the final project, not for AMIKOM itself.

Based on the users, this application has 3 types of users, namely admin, lecturer, and student.
Features for admin include:
- View all user data, classes, materials, forum posts, and website visit graphs
- Verifying new user data
- Changing and deleting user data
- Suspend classes,
- Delete post and materials

The features for lecturer include:
- Create a new class,
- Posting assignments or materials in class,
- Provide an assessment of student assignments,
- Upload and edit new material (public - visible to all students),
- Make a post on the forum,
- Comment in the forum,
- Delete a post or comment,
- Editing information and deleting accounts,

The features for student include:
- Has a unique score and predicate based on activity,
- Join a new class using code,
- Submit assignments in class,
- Comment on class or forum posts,
- Make a post on the forum,
- Delete a post or comment,
- View the list of lecturers,
- View a list of lecture materials from all lecturers and download them,
- Editing information and deleting accounts,

Frontend Tools:

  • HTML
  • CSS
  • JavaScript

Backend Tools:

  • PHP
  • mySQL
